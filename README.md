# Бот “Організатор подій”

## Мета
Створити бота-помічника, який допомагає студенту організовувати події (лекції, зустрічі,
іспити, нагадування тощо), працюючи з датами та текстовими запитами.

## Основні функції
- Додавання подій
- Перегляд усіх подій
- Пошук за категорією
- Події на тиждень
- Видалення подій
